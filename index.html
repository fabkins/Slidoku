How to Play

This is a sliding block puzzle where each row and column needs to add up to 30. The 0 tile represents the empty space, and you can slide any adjacent piece into it.

Look for these visual cues:

• Tiles with a green border are in their correct final position
• Yellow tiles are fixed and cannot be moved

There are 3 difficulty levels, Easy, Medium, and Hard, games get progressively more challenging on difficulty level.

• Easy allows the board to be revealed.
• Medium has no reveal but a row has not been shuffled.
• Hard has no reveal, all rows have been shuffled and has two fixed tiles.
